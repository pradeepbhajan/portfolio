# Social Media Manager Portfolio

## 🚀 Deploy to Netlify (Free)

### Option 1 — Drag & Drop (Easiest)
1. Go to [netlify.com](https://netlify.com) and sign up free
2. Click **"Add new site" → "Deploy manually"**
3. Drag your entire project folder onto the page
4. Done! You get a live URL like `yourname.netlify.app`

### Option 2 — GitHub (Recommended for updates)
1. Push this folder to a GitHub repo
2. Go to [netlify.com](https://netlify.com) → "Add new site" → "Import from Git"
3. Connect your GitHub repo
4. Build settings: leave blank (static site)
5. Click **Deploy** — every push auto-deploys

### Option 3 — Netlify CLI
```bash
npm install -g netlify-cli
netlify login
netlify deploy --prod --dir .
```

## ✏️ Customize
- Replace `Your Name` in `index.html` with your actual name
- Update email, WhatsApp number, LinkedIn URL in the contact section
- Upload your photo in the About section (click the photo area)
- Upload portfolio screenshots in the Portfolio section
- Update case study numbers with your real results

## 📁 Files
- `index.html` — main page
- `style.css` — all styles
- `script.js` — interactions, portfolio upload, lightbox
- `netlify.toml` — deploy config
